---
title: 'BasicRunner'
description: 'Single entry-point execution model for aerpawlib scripts.'
---

`BasicRunner` is the simplest execution model. You subclass it, decorate one async method with `@entrypoint`, and the framework calls that method after connecting and arming the vehicle.

## How it works

When the launcher starts a `BasicRunner`:

1. It constructs the vehicle and waits for connection.
2. It calls `runner.run(vehicle)`, which discovers and invokes the `@entrypoint` method.
3. When the entry point returns, `cleanup()` is called automatically.

## Subclassing BasicRunner

```python
from aerpawlib.v1 import BasicRunner, entrypoint, Drone, Coordinate

class MyMission(BasicRunner):
    @entrypoint
    async def run(self, drone: Drone):
        await drone.takeoff(10)

        target = Coordinate(35.7275, -78.6960, 10)
        await drone.goto_coordinates(target)

        await drone.land()
```

## The `@entrypoint` decorator

```python
from aerpawlib.v1 import entrypoint
```

Mark exactly one async method with `@entrypoint`. The decorated method must accept `self` and a `Vehicle` argument:

```python
@entrypoint
async def run(self, vehicle: Vehicle) -> None:
    ...
```

<Warning>
A `BasicRunner` subclass must have exactly one `@entrypoint`. Defining zero raises `NoEntrypointError`; defining more than one raises `StateMachineError`.
</Warning>

## Custom CLI arguments

Override `initialize_args` to parse additional command-line arguments:

```python
from typing import List

class MyMission(BasicRunner):
    def initialize_args(self, args: List[str]) -> None:
        import argparse
        parser = argparse.ArgumentParser()
        parser.add_argument("--altitude", type=float, default=10)
        parsed, _ = parser.parse_known_args(args)
        self.altitude = parsed.altitude

    @entrypoint
    async def run(self, drone: Drone):
        await drone.takeoff(self.altitude)
        await drone.land()
```

## Cleanup

Override `cleanup()` to release resources when the script exits:

```python
class MyMission(BasicRunner):
    def cleanup(self) -> None:
        print("Mission complete, releasing resources")

    @entrypoint
    async def run(self, drone: Drone):
        await drone.takeoff(10)
        await drone.land()
```

## Full example with imports

```python
from aerpawlib.v1 import (
    BasicRunner,
    entrypoint,
    Drone,
    Coordinate,
    VectorNED,
    SafetyCheckerClient,
)
import asyncio

class FullMission(BasicRunner):
    def __init__(self):
        self.checker = SafetyCheckerClient("127.0.0.1", 14580)

    def cleanup(self) -> None:
        self.checker.close()

    @entrypoint
    async def run(self, drone: Drone):
        ok, msg = self.checker.check_server_status()
        if not ok:
            print(f"Safety checker offline: {msg}")
            return

        ok, msg = self.checker.validate_takeoff_command(
            10, drone.position.lat, drone.position.lon
        )
        if not ok:
            print(f"Cannot take off: {msg}")
            return

        await drone.takeoff(10)

        waypoints = [
            Coordinate(35.7275, -78.6960, 10),
            Coordinate(35.7280, -78.6955, 15),
        ]
        for wp in waypoints:
            ok, msg = self.checker.validate_waypoint_command(drone.position, wp)
            if ok:
                await drone.goto_coordinates(wp)
            else:
                print(f"Skipping {wp}: {msg}")
                break

        await drone.land()
```

## Running the script

```bash
aerpawlib --script my_mission.py --vehicle drone --conn udp://:14540
```
